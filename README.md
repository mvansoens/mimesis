# mimesis

Mimesis is a Swift and SwiftUI learning app that teaches basic math through music theory, and music theory through basic math.

## Vision

Build an interactive experience where learners hear a musical concept, manipulate it, and then connect it to the underlying math.

## MVP

- Feature: Interval Trainer
- User adjusts one tone to match a target interval by ear
- App reveals the ratio (example: 3:2) and interval name (example: perfect fifth)
- Learning outcomes: ratios, listening skills, and pattern recognition

## Roadmap Snapshot

- Level 1: fractions through rhythm
- Level 2: ratios through harmony
- Level 3: exponents through pitch
- Level 4: waves and superposition intuition

## Tech Direction

- UI: SwiftUI
- Audio engine: AVAudioEngine or AudioKit
- Visuals: SwiftUI Canvas and Core Animation
